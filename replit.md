# בוט דיסקורד

## סקירה כללית
פרויקט זה הוא בוט דיסקורד שנבנה עם Node.js ו-Discord.js. הבוט כולל מערכת פקודות גמישה ויכולת לשלוח הודעות לערוצים שונים.

## תכונות עיקריות
- מערכת פקודות עם prefix `!`
- 5 פקודות בסיסיות: help, ping, info, send, echo
- שליחת הודעות לערוצים ספציפיים
- לוגים מפורטים של פעילות הבוט
- ניהול שגיאות מתקדם

## מבנה הפרויקט
```
/
├── index.js           # קובץ ראשי של הבוט
├── commands/          # תיקיית פקודות
│   ├── help.js       # רשימת פקודות
│   ├── ping.js       # בדיקת זמן תגובה
│   ├── info.js       # מידע על הבוט
│   ├── send.js       # שליחת הודעות לערוצים
│   └── echo.js       # חזרה על הודעות
├── package.json       # תלויות הפרויקט
├── .env.example       # דוגמת קובץ סביבה
└── README.md          # תיעוד מלא
```

## טכנולוגיות
- **Node.js 20**: סביבת ריצה
- **Discord.js 14**: ספריית Discord הרשמית
- **dotenv**: ניהול משתני סביבה

## הגדרות נדרשות
1. טוקן בוט מ-Discord Developer Portal
2. הפעלת MESSAGE CONTENT INTENT בפורטל
3. הוספת הבוט לשרת עם ההרשאות הבאות:
   - Send Messages
   - Read Messages/View Channels
   - Read Message History
   - Manage Messages

## הוספת פקודות חדשות
כל פקודה היא קובץ JS נפרד בתיקיית `commands/` עם המבנה:
```javascript
module.exports = {
  name: 'command-name',
  description: 'תיאור',
  execute(message, args, client) {
    // לוגיקת הפקודה
  },
};
```
