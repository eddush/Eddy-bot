# 🤖 בוט דיסקורד

בוט דיסקורד עם מערכת פקודות ויכולת שליחת הודעות.

## 📋 תכונות

- ✅ מערכת פקודות גמישה עם prefix `!`
- ✅ שליחת הודעות לערוצים
- ✅ פקודות בסיסיות: help, ping, info, send, echo
- ✅ לוגים של כל פעילות הבוט
- ✅ ניהול שגיאות מתקדם

## 🚀 התקנה והפעלה

### שלב 1: קבלת טוקן בוט מדיסקורד

1. היכנס ל-[Discord Developer Portal](https://discord.com/developers/applications)
2. לחץ על "New Application" וצור אפליקציה חדשה
3. בחר שם לבוט שלך ולחץ "Create"
4. עבור לטאב "Bot" בתפריט הצד
5. לחץ על "Add Bot" ואשר
6. תחת "TOKEN", לחץ על "Reset Token" ולאחר מכן "Copy" כדי להעתיק את הטוקן
7. **שמור את הטוקן במקום בטוח!** לא תוכל לראות אותו שוב

### שלב 2: הפעלת ה-Intent הנדרש

1. בעמוד הבוט, גלול למטה ל-"Privileged Gateway Intents"
2. הפעל את האפשרות "MESSAGE CONTENT INTENT"
3. שמור את השינויים

### שלב 3: הוספת הבוט לשרת

1. עבור לטאב "OAuth2" > "URL Generator"
2. ב-"SCOPES", בחר:
   - `bot`
3. ב-"BOT PERMISSIONS", בחר:
   - `Send Messages`
   - `Read Messages/View Channels`
   - `Read Message History`
   - `Manage Messages` (אם תרצה שהבוט ימחק הודעות)
4. העתק את ה-URL שנוצר בתחתית העמוד
5. פתח את ה-URL בדפדפן והוסף את הבוט לשרת שלך

### שלב 4: הגדרת הפרויקט

1. צור קובץ `.env` בתיקייה הראשית:
```bash
DISCORD_TOKEN=הטוקן_שהעתקת_משלב_1
CLIENT_ID=ה-ID_של_הבוט
```

2. ודא שהתוכנה מותקנת:
```bash
npm install
```

3. הפעל את הבוט:
```bash
npm start
```

## 📚 פקודות זמינות

| פקודה | תיאור | דוגמה |
|-------|-------|-------|
| `!help` | מציג את רשימת הפקודות | `!help` |
| `!ping` | בדיקת זמן תגובה | `!ping` |
| `!info` | מידע על הבוט | `!info` |
| `!send` | שליחת הודעה לערוץ | `!send #general שלום!` |
| `!echo` | הבוט חוזר על ההודעה | `!echo שלום עולם` |

## 🔧 הוספת פקודות חדשות

כל פקודה נמצאת בקובץ נפרד בתיקיית `commands/`. כדי להוסיף פקודה חדשה:

1. צור קובץ חדש בתיקיית `commands/`, למשל: `mycommand.js`
2. השתמש בתבנית הבאה:

```javascript
module.exports = {
  name: 'mycommand',
  description: 'תיאור הפקודה',
  execute(message, args, client) {
    message.reply('התגובה שלך כאן');
  },
};
```

3. הבוט יטען את הפקודה החדשה אוטומטית בהפעלה הבאה

## 🛠️ טיפולוגיה

### הבוט לא מתחבר

- ✅ וודא שהטוקן בקובץ `.env` תקין
- ✅ ודא שהפעלת את "MESSAGE CONTENT INTENT" בפורטל Discord
- ✅ בדוק שאין שגיאות בקונסול

### הבוט לא מגיב לפקודות

- ✅ וודא ש-"MESSAGE CONTENT INTENT" מופעל
- ✅ ודא שהבוט נמצא בשרת
- ✅ בדוק שאתה משתמש ב-prefix הנכון (`!`)
